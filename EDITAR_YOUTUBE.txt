EDITAR LINK Y TÍTULO DEL VIDEO (sin tocar el HTML)

1) Abrí el archivo: youtube-config.js
2) Reemplazá SOLO estos campos:

- youtubeUrl: "..."  (pegá acá cualquier link de YouTube: youtu.be / watch?v= / embed / shorts)
- title:      "..."  (título del video para accesibilidad)

✅ Listo. El sistema extrae el ID del video y arma:
- el botón “Abrir en YouTube”
- el reproductor embebido (iframe)

Opcional:
- thanks:   texto principal del modal
- subtitle: texto secundario
- linkText: texto del link en la pantalla de vertidos

NOTA IMPORTANTE
Si abrís el HTML como archivo (file://) en tu compu, YouTube suele bloquear el reproductor y puede aparecer un error.
Cuando lo subís a GitHub Pages (https://...) el embed funciona.
